# Soporte

Esta carpeta agrupa todo lo que no es la lectura principal de la entrega, pero sirve para reproducibilidad, auditoria y trazabilidad.

- `codigo/`: notebooks auxiliares del pipeline.
- `Submits/`: submits historicos usados por la reconstruccion competitiva.
- `otros csv/`: features procesadas, lookup, columnas y archivos del solver.
- `modelos/`: metadatos de configuracion y entrenamiento.
- `docs/`: bitacora completa del proceso experimental.
